  • Stylish eye catching colors with easy configuration and nice animations.
  • Variety number of charts supported. Tile, Vertical Bar, Horizontal Bar, Pie, Doughnut, Line.
  • Easily Movable with Configurable Size.
  • Enjoy Full screen mode for particular charts.
  • Update Refresh interval for data updates.
  • Different Range of Colors available for all charts. There are 450+ selectable colors.
  • Easy Domain interface that allows you to filter records for your Dashboards.
  • Supported Three types of Operations for your charts.
  • Configure your charts with Group by and different Measures with Date filter field.
  • Supported 10+ Date filters. Such as Last month, This month, Next month etc. Also supported custom date range.
